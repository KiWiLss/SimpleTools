# SimpleTools

#1,日志使用
        1,在application中可以设置是否打印日志
          L.setIsLog(false);
          设置打印的TAG
          L.setTag("hfiehi");
         2,一般性的使用
         LL.e("222222222");
          LL.i("");
         LL.d("");
         3,打印json和xml
          L.json("");
          L.xml("");
          4,将任意类型的数据打印成String
            L.ii("");
            L.ee("");
