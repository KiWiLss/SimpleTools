# SimpleTools

#1,日志使用<br>
        1,在application中可以设置是否打印日志<br>
          L.setIsLog(false);<br>
          设置打印的TAG<br>
          L.setTag("hfiehi");<br>
         2,一般性的使用<br>
         LL.e("222222222");<br>
          LL.i("");<br>
         LL.d("");<br>
         3,打印json和xml<br>
          L.json("");<br>
          L.xml("");<br>
          4,将任意类型的数据打印成String<br>
           L.ii("");<br>
           L.ee("");
